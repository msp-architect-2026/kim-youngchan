# DropX — Kubernetes Manifests v2

> 한정판 운동화 드롭 상황에서 동시성 충돌을 방지하고 재고 음수 없이 주문을 처리하는  
> k3s + GitOps 기반 초고트래픽 주문 인프라

## ⚠️ 핵심 주의사항

| 항목 | 내용 |
|------|------|
| **HTTP Registry** | k3s registries.yaml insecure 설정 필수 (01번 가이드) |
| **PV 불필요** | local-path가 PVC 생성 시 자동으로 PV 생성 |
| **Secret 보안** | `secret.yaml` Git 커밋 금지. 적용 후 값 되돌리기 |
| **k6 위치** | 반드시 PC #2에서 실행 (측정 왜곡 방지) |
| **이미지 태그** | 운영 시 latest 금지. CI에서 $CI_COMMIT_SHORT_SHA 사용 |
| **단일 노드** | PDB는 drain 블로킹 가능. kill-pod 데모는 정상 동작 |
